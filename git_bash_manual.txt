※下記の手順はすべて「Git bash」インストール済みを前提とする
※[]でくくった内容はgit bashへの入力コマンド

1.リポジトリのローカルクローン
　1_1.任意のフォルダを選択し、右クリックメニューから「Open Git Bash here」を選択
　
　1_2.[git clone (リモートリポジトリのURI)] を入力

2.ファイル追加
　2_1.[cd ./(リポジトリ名)] を入力し、フォルダ内に移動
　
　2_2.指定フォルダにファイルを追加する。
　
　2_3.[git add (ファイル名)] を入力
　
　2_4.[git commit -am "(更新コメント内容)"] を入力
　
　2_5.[git push origin (ブランチ名)] を入力
　※git hubにてブランチを一つも作成していない場合、push時に作成されるためmainブランチを推奨
　
　2_6.必要であれば、プルリクエストを作成する。

3.ファイルの更新
　3_1.[git checkout (ブランチ名)] を入力
　※develop等、作業用ブランチを切る元のブランチを選択
　
　3_2.[git pull origin (ブランチ名)] を入力
　※pullすることで、現在のリモートブランチの状況を取得する
　
　3_3.[git checkout -b (作業ブランチ名)] を入力
　※作業ブランチ名は任意。「feature/日付」等。
　
　3_4.ファイル内容の更新を行う
　
　3_5.[git commit -am "(更新コメント内容)"] を入力
　
　3_6.[git push origin (作業ブランチ名)] を入力
　※リモートリポジトリ上に自動的にブランチが作成される
